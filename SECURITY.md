# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly using one of the following methods:

- **GitHub Private Reporting:** Use [GitHub's private vulnerability reporting](https://github.com/leakingmemory/housingplanner/security/advisories/new) to submit a report directly on this repository.
- **Email:** Send details to Jan-Espen Oversand at [sigsegv@radiotube.org](mailto:sigsegv@radiotube.org).

Please include as much detail as possible — steps to reproduce, affected versions, and potential impact — so the issue can be assessed and addressed quickly.

## Response

We will acknowledge receipt of your report and aim to provide an initial assessment as soon as possible. Security fixes will be released as patch updates.

## Scope

This policy applies to the housingplanner application and its build/release infrastructure maintained in this repository.
